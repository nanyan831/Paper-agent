# Paper Agent MVP 试用指南

## 五分钟上手

### 1. 启动项目

```powershell
cd Paper-agent
.\.venv\Scripts\activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

浏览器打开：**http://127.0.0.1:8000**

### 2. 配置 API Key

首次打开页面，顶部会显示 readiness 状态提示：

- ❌ 没有 API Key → 请先配置
- ✅ API Key 已配置 → 可以继续

配置方法：

1. 复制 `.env.example` 为 `.env`
2. 编辑 `.env`，填入你的 DeepSeek API Key：
   ```env
   DEEPSEEK_API_KEY="你的真实 key"
   ```
3. 重启服务

### 3. 导入论文 PDF

1. 点击左侧导航「爬虫控制」
2. 选择「导入本地 PDF」
3. 选择一篇或多篇论文 PDF
4. 等待导入完成

导入成功后会显示：
- 论文标题
- Paper ID
- 页数
- Chunk 数量

> 💡 提示：PDF 需要有可复制的文本。扫描版 PDF 目前会提示解析失败。

### 4. 打开论文阅读器

1. 点击左侧导航「探索文献」
2. 在文献列表中找到刚导入的论文
3. 点击「打开阅读器」

阅读器功能：
- 页码跳转
- 缩放控制
- 原文 / 译文 / 对照 三种阅读模式
- 阅读器内「查出处」

### 5. 提问并获得来源

1. 点击左下角的「AI 对话」按钮
2. 输入一个关于论文内容的问题，例如：
   - "这篇论文的核心方法是什么？"
   - "实验部分用了哪些数据集？"
   - "结论部分有什么发现？"

回答特点：
- 每个关键结论都附有来源
- 来源显示：论文标题、页码、原文片段
- 点击来源可以直接跳转到 PDF 对应页码

### 6. 查看 Token 使用情况

系统内置每日 Token 预算保护（默认 200,000 tokens/天）：

- 每次回答后显示本次 Token 消耗
- 预算用尽时会明确提示
- 第二天自动重置

## 常见问题

### Q: 为什么说"本地论文库没有找到可引用依据"？

A: 可能的原因：
1. 还没有导入任何 PDF → 先导入论文
2. 问题和已导入的论文内容不相关 → 换个问题或导入相关论文
3. PDF 是扫描版，没有可解析的文本 → 目前只支持文本型 PDF

### Q: Token 预算用尽了怎么办？

A: 
1. 等待第二天自动重置
2. 或者在 `.env` 中调整 `DAILY_TOKEN_BUDGET` 数值
3. 建议：问题尽量具体，避免宽泛提问

### Q: 来源的"证据强弱"是什么意思？

A: 
- **high**：有明确页码、检索分数高、相关性强
- **medium**：有页码但分数一般
- **low**：相关性较弱，只能作为线索
- **unknown**：缺少分数或页码信息

### Q: 为什么回答开头会写"证据不足，只能作为线索"？

A: 当来源少于 2 个或有低置信度来源时，系统会自动加上这个提示。这是可信度保护机制，避免给出过于确定的结论。

## 核心特性总结

✅ **本地优先**：所有论文和聊天记录保存在本地  
✅ **来源可追溯**：每个结论都能回到原文页码  
✅ **Token 可控**：每日预算保护，单次回答 3000-8000 tokens  
✅ **无需理解 RAG**：开箱即用，不用关心向量库细节
